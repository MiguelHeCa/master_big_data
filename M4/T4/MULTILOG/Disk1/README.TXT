A. General information
   
1) You can create a desktop icon for MULTILOG using Windows Explorer.
   Locate the directory to which you have installed  MULTILOG.
   Right click on MLGWIN.EXE and select the Send To > Desktop (create shortcut)
   option from the pop-up menu.
